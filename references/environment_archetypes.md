## AT-01: 対面構築

### 典型構成

メガ進化 + 物理瞬間火力ポケで対面で取り合う。サイクルしない、削り合いしない、純粋に対面で殴り勝つ。

代表ポケ:
- **メガガルーラ** (おやこあい連続2回攻撃、ねこだまし起点)
- **メガロップ** (とびひざ / トリプルアクセル / インファ)
- **メガミミロップ** (上同)
- **メガカイリュー** (しんそく + マルスケ)
- **メガルカリオ** (はどうだん / インファ)
- **ガブリアス** (タスキ + じしん高火力)

### 戦術特徴

- 先発でいきなりメガ進化、即火力
- 受けポケを置かない、6 体全員アタッカー
- ステロ撒き役は採用するが、サポートは最小限

### 警戒すべき技 (追加効果重視)

- **ねこだまし** (フリンチ 100%、初動取られる) ← メガガル T1 必発
- **とびひざげり** (失敗時 自分 1/2 削り、相手読みリスク技)
- **しんそく** (先制 +2、終盤詰め)
- **冷凍パンチ** (氷 4 倍ガブ / メガカイ即死、やけど 10%)
- **オーバーヒート** (C-2 で次ターン引き、燃やす可能性 10%)

### 想定選出 (こちら側の対応指針)

物理受け 1 + 特殊受け 1 + アタッカー 1 が基本。
特に「ねこだまし無効化手段」(せいしんりょく持ち or ゴースト鋼複合) を必ず入れる。

### 対 AT-01 の鍵

```
1. 先発でメガ進化させない
2. ねこだまし無効化 (せいしんりょく / ゴースト)
3. しんそく耐性 (ゴースト無効 / マルスケ)
4. 持久戦に持ち込めば有利
```

## AT-02: サイクル構築

### 典型構成

役割を分担した 6 体で交代しながら相手を削る。受けと攻めを循環させて時間をかけて勝つ。

代表ポケ:
- **アシレーヌ** (めいそう型、特殊軸)
- **アーマーガア** (てっぺきボディプ、物理受け)
- **カバルドン** (砂 + ステロ、起点作り)
- **ハラバリー** (特殊電気水)
- **ロトムウォッシュ** (10万 + ハイドロ + でんじは)
- **メガフシギバナ** (やどりぎ + 草毒)

### 戦術特徴

- 引きを多用、対面取らずに有利対面作り
- ステロ + 起点作りでじわじわ削る
- ダブルダメージ (毒 + ステロ + やどりぎ) で時間切れ狙い

### 警戒すべき技 (追加効果)

- **やどりぎのタネ** (Champions ナーフ後 1/16/T 削り、累積で死ぬ)
- **どくどく** (もうどく、累積 1/16, 2/16, 3/16...)
- **でんじは** (S 半減 + 麻痺行動率 12.5%)
- **ステルスロック** (交代時 タイプ別ダメ、岩 4 倍リザY 50% 削り)
- **あくび** (次T眠り、相手の引き強制)

### 対 AT-02 の鍵

```
1. ちょうはつ持ちで起点作り阻害
2. 受け崩し役 (毒持ち / かげふみ) で物理サイクル止める
3. ステロ撒く前に潰す
4. ねがい / なまける 等の回復ループを どくどく で台無しに
```

## AT-03: 受けループ

### 典型構成

純粋な受けポケ 6 体、攻撃力低めで HP / 防御特化。相手を削りきれず時間切れ狙い。

代表ポケ:
- **アーマーガア** (てっぺきボディプ + はねやすめ)
- **メガフラエッテ (フラージェス)** (永遠の花、特殊受け)
- **ピクシー** (マジックガード / すりぬけ)
- **ブラッキー** (ねがいごと + どくどく)
- **メガフシギバナ** (やどりぎ + ねむりごな)
- **ヌオー** (てんねん + どくどく)

### 戦術特徴

- 攻撃技は ボディプレス / イカサマ / どくどく / イワオコシ 等の汎用技
- 状態異常 + 削りで時間切れ
- 物理 / 特殊 / 状態異常 すべての受け方を用意

### 警戒すべき技

- **てっぺき** (B+2、ボディプレス火力跳ね上げ)
- **ボディプレス** (B 依存ダメージ、てっぺき 2 回で確 1 ライン)
- **やどりぎのタネ + どくどく** (ダブルダメージで強制ターン)
- **はねやすめ** (HP 1/2 回復 + 1T 飛行消失で氷半減)

### 対 AT-03 の鍵

```
1. 状態異常無効化 (マジカルリーフ + ラム持ち)
2. 受け崩し役 (メガゲンガー かげふみ / メガミミロ ノ / メガフラ ムンフォ)
3. ちょうはつで状態異常技封じ
4. 火力ゴリ押し (りゅうせいぐん / 大文字 / きあいだま)
5. 時間切れ覚悟で引き分けでも上等
```

## AT-04: 晴れパ

### 典型構成

晴れ天候を生かす火力構築。

代表ポケ:
- **メガリザードンY** (晴れ ソーラービーム 1T / 大文字 1.5 倍)
- **コータス** (ひでり自動展開、晴れ起点役)
- **メガフシギバナ** (晴れ ようりょくそ S 2 倍)
- **メガリザードンX** (晴れ補助、メガ後 ドラゴンタイプ)
- **キュウコン** (ひでり、起点)

### 戦術特徴

- T1 ひでり始動 → メガリザY 即火力
- ソーラービーム 1 ターン化で水/地受けを蒸発
- 大文字 / オーバーヒート 1.5 倍で鋼即死

### 警戒すべき技

- **ソーラービーム** (Champions では晴れ下 1T 化、威力 120 で受け破壊)
- **大文字 1.5 倍** (晴れ下メガリザY なら 165 相当、ほぼ確 1)
- **オーバーヒート 1.5 倍** (晴れ下なら確 1 連発、C-2 はあるが)
- **ねっぷう** (フィールド全体、Singles でも有効)

### 対 AT-04 の鍵

```
1. メガリザY を即落とし (ガブ がんせきふうじ岩 4 倍 確 1)
2. カバ で天候上書き (砂)
3. 水技で削り (アシレ ハイドロ / メガカイ 10万)
4. ステロ 50% 削りで即落とし圏内に
```

## AT-05: 積み構築

### 典型構成

クエスパトラ / ノココッチバトンパス / メガミミロ剣舞 など、積み技起点でゲーム決める構築。

代表ポケ:
- **クエスパトラ** (ルミナコリジョン C+, S+ 起点)
- **ノココッチ** (バトンパスで S+, A+ 継承)
- **メガミミロップ** (剣舞 + フェイント)
- **メガカイリュー** (龍舞)
- **ガブ** (剣舞 + スカショ S+)

### 戦術特徴

- 起点作り役 (ステロ持ち) を T1 排除
- 積みポケで一気に積む
- 積み後にスイープ

### 警戒すべき技

- **ルミナコリジョン** (C+ 確定、命中 100%)
- **バトンタッチ** (積みステ継承、相手構築読み難)
- **つるぎのまい** (A+2)
- **りゅうのまい** (A+1, S+1 同時)

### 対 AT-05 の鍵

```
1. ちょうはつで積み技封じ
2. ふきとばし / ほえる / ドラゴンテール で強制交代 (積みリセット)
3. 急所狙い (急所はランク補正無視)
4. みちづれで積みエース道連れ
5. はたきおとす / トリック で持ち物剥奪
```

## AT-06: トリル構築

### 典型構成

低 S 高耐久ポケでトリックルームを展開、5 ターン S 逆転して殴る。

代表ポケ:
- **スターミー** (現環境では希薄)
- **ハバタクカミ** (S 高いが、トリル下では遅いポケが優位)
- **ガチグマ アカツキ** (S 52、トリル下で抜く)
- **メガジュペッタ** (トリル + みちづれ)

### 戦術特徴

- T1 トリル起動 → 5 ターン S 逆転
- 高 S アタッカーが上から取られる

### 対 AT-06 の鍵

```
1. ちょうはつでトリル封じ
2. トリル下では受けに回る (5 ターン待つ)
3. 急所 / みちづれ でターン消費
4. しんそくは優先度技なので S 逆転に影響されない (積極使用)
```

## AT-07: 中堅構築

### 典型構成

メガ無し中堅レート帯で多い、特定の 1-2 体だけ強くて残りはサポート。

特徴:
- メガストーン採用率低い
- ハバタクカミ + ガブ + サポート 4 体 のような分散型
- ハイパー帯までは多いが、マスター帯では減る

### 対 AT-07 の鍵

```
1. メガ採用 = こちら火力アドバンテージ
2. 受けで凌いで火力勝ち
3. 油断厳禁 (個別ポケが強い場合あり、特にハバタクカミ)
```

---

## AT-08: ステロループ / 特殊構築

### 典型構成

- ステロ + 強制交代 (ふきとばし / ドラゴンテール) ループでハザードダメ蓄積
- ねむねごガード型 (ねむる + ねごと + 状態異常無効)
- 凍える風 + ねっぷう ダブル天候型 (Singles では希薄)

### 警戒すべき技

- **ふきとばし** (優先度 -6、強制交代でステロダメ)
- **ドラゴンテール** (威力あり + 強制交代)
- **ほえる** (同上、変化技)
- **トリック** (持ち物入れ替え、こだわり押し付け)

### 対 AT-08 の鍵

```
1. ステロ前に潰す
2. ねむり技持ちには ラムのみ / 状態異常無効持ち
3. トリック注意 (こちらのスカーフを押し付けられる)
```

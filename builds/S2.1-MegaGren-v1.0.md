## §4 構造的弱点 + 改善提案 (2026-06-05 実数値で再評価)

> ★ 旧版の「ブリジュ詰み/マスカ抜かれ→カビゴンに差し替え」は **標準252/252仮定の誤算**だった。
> 実数値(rawStats)で計算し直すと、4大脅威(ブリジュ/リザ/イダイ/マスカ)は**現6体で全て勝てている**。
> **アーマーガアの地面無効は地震/ガブ飽和環境の核資産 → 残す。** 差し替えは不要。

### 4大脅威の正しい対処 (実数値 calc)
- **ブリジュ(主流無振Modest)**: 初手オオニューラ インファ **110-130%確1**。Bold物理受(少数)のみ2HKO→ステロ+2発
- **マスカ**: 初手オオニューラがトリフラ急所も耐え(38-45%)→**DireClaw 135-158%確1**
- **イダイ**: メガゲッコ草結び **121-144%確1** / スカガブ地震2HKO
- **リザY**: スカガブ岩雪崩 **151-180%確1** (晴れ展開前に処理)

### 残る本物のソフトスポット (差し替え不要、運用でケア)
1. **メガゲッコは ひかえめ S183 の低速breaker** → 素マスカ/スカーフ/ドラパルトに抜かれる。タンク&ブレイク運用、revengeはスカガブに任せる
2. **防御特化ブリジュ(Bold/Calm 少数)** は1発で落ちない → ステロ+ドラゴンテール流しで削る、メガゲン罠は流星群被せに注意
3. **中盤スカーフマスカ(S288)** はオオニューラ退場後に全抜き圏 → オオニューラを温存して当てる / スカガブ呼ぶ
4. **ハッサム#14**: 炎技ゼロで割りにくい → スカガブ地震+メガゲッコで負荷

### 改善提案 (任意・低コスト、骨格維持)
1. **スカガブ ステロ → ほのおのキバ** (こだわり死に技の解消 + 炎ゼロの穴埋め)。フェアリーはオオニューラ毒で確1済=どくづきは不要。炎はハッサム/ナットレイを2発で割り、地面が透かされるエアームド/対面アーマガに触れる唯一の技 ← 唯一の推奨変更
2. (好みで) メガゲッコ くさむすび → みずしゅりけん (先制でS183の遅さを補完、ただしカバ確1力↓)
3. 構成は完成度高く、メンバー差し替えは非推奨

### 序盤 (T1-3)

- メガ進化判断: 相手のメガ確定後に進化先を決める (高速→メガゲッコ / 受け→メガゲン)

- メガゲン滅び: みがわり+まもるで3ターン稼ぎ、かげふみ拘束で1:1以上を取る
- スカガブで上から詰め、メガゲッコで全抜き

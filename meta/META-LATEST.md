## §4 自構築 S2.1 の構造的弱点 + 改善提案

### 4.1 ★最優先: スカーフガブの「ステルスロック」は死に技

```
こだわりスカーフは初手選択技に拘束される。ステロを撒いた瞬間ステロに拘束 = 起点化される。
ステロ撒き役は ブリジュ (非拘束) に一本化済 → スカガブのステロ枠は不要。
→ 推奨: ステロ を **ほのおのキバ** に変更 (現状 炎技ゼロの穴を埋める: ハッサム メガ54-65%確2/ナットレイ55-66%確2(炎4倍)、地面が透かされるエアームド・対面アーマガに触れる唯一の技)
   ※どくづき(フェアリー)はオオニューラ フェイタルクロー(毒)と役割被りで不要
```

### 4.2 鋼受け 2 枚の天敵 = 炎・晴れ

```
ブリジュ(鋼/竜) + アーマガ(飛/鋼) は共に炎 2倍。晴れメガリザY が両受けを溶かす:
  リザY オバヒ → アーマガ 215-255% 確1
  リザY ねっぷう(晴) → ブリジュ 特化D でも 68-80% 確2
対策: スカガブ岩雪崩がリザY確1 (151-179%) → 晴れ展開前にスカガブで即処理が必須ライン
```

### 4.3 メガ被り (ゲッコウガナイト + ゲンガナイト) の選出規律

```
1試合1メガ。3体選出でメガ2枚を同時投入するとどちらか1枚は素 (= タスキ無し裸ゲンガー等)。
推奨: 高速オフェンス相手 → メガゲッコ進化 / サイクル・受け相手 → メガゲンガー進化。両者の同時選出は基本回避。
```

### 4.4 ハザード除去なし → SR 弱点メンバーに負担

```
高速スピン/コートチェンジ/はたきおとし 不在。
アーマガ(飛) はステロ 25% 被弾、メガゲッコ(紙耐久) も削れて行動回数が減る。
許容範囲だが、対ステロ構築では アーマガに とんぼ返り より はねやすめ温存で延命を意識。
```

### 4.5 差し替え候補 (Champions M-2 適合・任意)

| 候補 | 役割 | 入れ替え対象 | trade-off |
|---|---|---|---|
| スカガブ ステロ→ほのおのキバ | 対鋼(ハッサム/ナットレイ)+地面透かし対応 | スカガブの死に技解消 | なし (純粋強化、どくづきは毒被りで不要) |
| ドヒドイデ / テッカグヤ | 特殊受け補強 (炎/フェアリー) | アーマガ or ブリジュ | サイクル力↑ だが攻撃面↓ |
| メガゲッコ みずしゅりけん採用 | 先制+タスキ/MS貫通 | くさむすび or じんつうりき | カバ/ギャラへの確1力↓ |
